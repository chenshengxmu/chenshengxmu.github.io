* Safari is the default web browser in both iOS and macOS, but did you know the latest iOS 11 and macOS High Sierra updates supercharge Safari’s Reader mode capabilities?
* In this piece, we’ll show you how you can configure Safari to automatically enable Reader mode every time you visit a specific website on both platforms.
* Subscribe to iDB on YouTube!
* If you’re using an iOS 11 device, all you have to do is follow these simple steps:
* 1) Launch Safari and visit a website.
* 2) Tap and hold on the Reader mode button in the URL bar:
![img](http://media.idownloadblog.com/wp-content/uploads/2017/10/Use-Reader-Mode-Automatically-Safari-iOS-11-1.jpg)
* 3) In the prompt that appears, tap on the Use on “<website name>” button.
* Now, whenever you launch a web page from the website you’ve configured this option for, Reader Mode will start up automatically for your convenience:
![img](http://media.idownloadblog.com/wp-content/uploads/2017/10/Reader-Mode-Automatically-Safari-iOS-11-2.jpg)
* If you wanted to disable this feature, just follow steps 1-2 again, but tap on the Stop Using on “<website name>” button instead, like so:
![img](http://media.idownloadblog.com/wp-content/uploads/2017/10/Stop-using-Reader-Mode-Safari-iOS-11.jpg)
* As we indicated earlier, the same feature exists in macOS High Sierra. To perform this action on your Mac, follow these steps instead:
* 1) Launch Safari and visit a website.
* 2) Click and hold on the Reader Mode button in the URL bar:
![img](http://media.idownloadblog.com/wp-content/uploads/2017/10/Reader-View-Safari-macOS-1.jpg)
* 3) In the pop-up that appears, select the Use Reader Automatically on “<website name>” option to select it:
![img](http://media.idownloadblog.com/wp-content/uploads/2017/10/Reader-Automatically-Safari-macOS-2.jpg)
* Safari in macOS High Sierra will now begin loading every web page associated with the website you’ve configured this option for in Reader Mode rather than standard mode:
![img](http://media.idownloadblog.com/wp-content/uploads/2017/10/Reader-View-Safari-Automatically-macOS-3.jpg)
* You can disable this feature by repeating steps 1-2 for a second time, and then clicking on the Use Reader Automatically on “<website name>” option to deselect it:
![img](http://media.idownloadblog.com/wp-content/uploads/2017/10/Turn-off-Safari-Automatic-Reader-Mode-macOS.jpg)
* As you’ve just witnessed, the steps to automatically force Reader mode on a website in iOS 11 or macOS High Sierra are as easy as 1-2-3. Now, you can browse the internet in peace – all the time.
